# Ollama News Agent

Ollama News Agent is an AI-powered news retrieval and summarization agent built using the Phi (phidata) agent framework and Ollama with the Mistral model.

## Features
- Uses a local LLM via Ollama (Mistral)
- Searches real-time news using Google Search
- Retrieves the latest news for a given topic
- Filters and returns the top 4 unique news items
- Supports both English and French news sources
- Outputs clean, markdown-formatted responses

## Tech Stack
- Python
- Phi (phidata) Agent Framework
- Ollama (Mistral)
- Google Search Tool

## Use Case
This project demonstrates how to build a local AI agent that combines tool usage and LLM reasoning to fetch and summarize current news efficiently.
